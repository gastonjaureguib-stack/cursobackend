# Agencia de Viajes - API Backend de Servicios y Reservas

## Descripción

Este proyecto es una API REST desarrollada con **Node.js**, **Express** y persistencia mediante **FileSystem (archivos JSON)**.

Luego de mirar la clase mil veces al fin el el sistema permite gestionar los servicios turísticos disponibles y las reservas realizadas por los clientes.

La información se almacena en archivos JSON, por lo que los datos permanecen guardados aunque el servidor se reinicie.

## Tecnologías utilizadas

- Node.js
- Express
- FileSystem (fs/promises)
- dotenv
- JavaScript ES Modules
